Цели 

- создать набор инструментов, позволяющий разбирать связи между интересными замечаниями в сюжетах фильмов, книг, видеоигр, для выявления скрытых авторских посылов и новых интересных замечаний.
- создать пакет в Python


Задачи

- реализовывать фичи в порядке приоритета:
  1 - сохранение и разворачивание обратно в консоль "проекта"
  2 - вывод в текстовом виде "на экран"
  3 - работа с замечаниями
  3.1 - объекты замечаний, - свойства замечаний
  4 - работа со связями
  5 - работа с текстом
  5.1 - сравнение текста - различные расстояния между текстовыми фраментами
  5.2 - поиск в тексте подстрок с расстоянием меньше чем заданное
  5.3 - поиск в множестве строк с расстоянием меньше чем заданное
  5.4 - поиск в тексте подстрок по пользовательским критериям
  5.5 - поиск в множестве строк по пользовательским критериям
  6 - работа с графами
  6.1 - хранение графа
  6.2 - выбор подграфа, сохранение в отдельный граф
  7 - работа с множествами
  7.1 - чёткие множества - пересечение множеств - объединение множеств - мощность множества - разность множеств - отношение множеств
  8 - подвыбор замечаний и связей
  8.1 - по критериям - схожесть текстов в свойствах и описании
